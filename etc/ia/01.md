## рџ› пёЏ Detailed File Specifications

### 1. `manifest.json` (The Core Config)

This file registers your extension, requests permissions, and enables Incognito support.

* **Permissions needed:** `tabs`, `tabGroups`, `storage`, `alarms`.
* **Incognito Requirement:** To make it work in Incognito, we set `"incognito": "spanning"` (or `split` if you want separate processes, but `spanning` allows the background script to see everything if the user grants permission in Chrome settings).

### 2. `background/background.js` & `groupingEngine.js`

The background script runs persistently in the ecosystem via alarms.

* **State Tracking:** Chrome tabs have a `lastAccessed` timestamp. The script will use a periodic `chrome.alarms` check (e.g., every 1 minute) to scan open tabs.
* **Grouping Engine Logic:**
1. **Filter:** Skip pinned tabs, already-grouped tabs, or excluded hosts based on user preferences.
2. **Map/Normalize Domains:** Group alternative hosts (like `www.cnn.com` and `www.cnn.us`) into a unified "virtual key" using a normalization map defined in options.
3. **Window Isolation:** Group tabs either per-window or globally depending on the configuration.
4. **Auto-Naming:** Run a Longest Common Prefix (LCP) algorithm on the titles of matching tabs. If $length \ge 7$, use it; otherwise, fall back to the domain name.
5. **Smart Coloring:** Maintain an array of available Chrome group colors (`grey`, `blue`, `red`, `yellow`, `green`, `pink`, `purple`, `cyan`, `orange`). Ensure the selected color is mathematically distant from adjacent groups.



### 3. `options/options.html` & `.js` (The Control Panel)

This is where the heavy configuration lives. It will read/write to `chrome.storage.sync`.

* **Inputs needed:**
* Delay threshold (number input + dropdown for minutes/hours/days).
* Textarea for excluded hosts (one per line).
* Textarea for grouped hosts aliases (e.g., `cnn.com, cnn.us`).
* Toggles for: Multi-window grouping, Including pinned tabs, Re-grouping already grouped tabs, and Auto-ungrouping on access.
* Minimum character threshold for string-matching titles (default: 7).



### 4. `popup/popup.html` & `.js` (The Quick Trigger)

A clean, small popup for everyday use.

* A "Blast Now" button to manually trigger the `tabehameha` grouping instantly instead of waiting for the timer.
* A quick status display (e.g., "Active: 4 groups managed").

## рџ’Ў Pro-Tips for this Specific Extension

* **Incognito Behavior:** Chrome strictly prevents mixing Incognito tabs and regular tabs in the same group. Your `groupingEngine.js` must treat `tab.incognito = true` tabs completely separately, even if they share a domain with a standard tab.
* **Host Extraction:** Use the `URL` API (`new URL(tab.url).hostname`) rather than regex to extract domains reliably.
* **The "Adjacent Color" Logic:** Because Chrome only supports 9 specific colors for tab groups, the best way to avoid adjacent visual blending is to loop through your target tabs' neighboring groups, check their colors, and pick the color from the Chrome color array that is furthest in index from the left and right neighbors.
